# 🛍️ Shopify Theme – Right Move Test

## 📌 Project Overview

This is a custom Shopify theme developed for testing and learning purposes.
It includes custom sections, responsive design, and performance-focused implementation.

---

## 🚀 Features

* Fully responsive design (mobile + desktop)
* Custom Shopify sections (Liquid based)
* Dynamic product rendering
* Infinite scroll / load more functionality
* Metaobject-based dynamic content
* Clean and maintainable code structure

---

## 🛠️ Tech Stack

* Shopify Liquid
* HTML5
* CSS3 / SCSS
* JavaScript (Vanilla JS)

---

## 📂 Folder Structure

* `/sections` → Custom Shopify sections
* `/snippets` → Reusable Liquid components
* `/assets` → CSS, JS, images
* `/templates` → Page templates

---

## ⚙️ Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/virendrasolanki03/test-theme-right-move.git
   ```
2. Upload theme to Shopify store
3. Customize via theme editor

---

## 💡 Key Implementations

* Featured product filtering using tags
* AJAX-based product loading
* Custom metaobject integration
* Pixel-perfect UI implementation
